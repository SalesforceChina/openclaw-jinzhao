## 微信文章导出任务

**目标文章：** https://mp.weixin.qq.com/s/LnbNX9kI3uDcDb57zdmrhg
**标题：** 冯矿伟：周一操作策略（0330）

### 尝试的方法
1. ✅ Docker 已安装 (29.3.1)
2. ❌ Docker 镜像 nulldream/wechat-article-exporter 不存在
3. ❌ pip 包 wechat-article-exporter 不存在
4. ❌ Jina Reader 返回 JavaScript 代码
5. ❌ 直接抓取需要浏览器环境

### 根本原因
微信文章内容通过 JavaScript 动态加载，需要：
- 完整浏览器环境（执行 JS）
- 正确的 Cookie 和 Referer
- 绕过反爬虫检测

### 推荐方案
1. **手动复制**（最快最可靠）- 在浏览器打开文章，复制粘贴
2. **浏览器插件** - 微信文章导出助手、MarkDownload
3. **在线服务** - 搜狗微信、wechatscope.com
4. **Playwright**（需要额外配置）- 自动化浏览器
