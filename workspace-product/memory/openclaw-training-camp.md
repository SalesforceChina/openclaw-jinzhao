# OpenClaw训练营 - 产品核心内容

## 图1：招生海报
AI正在重塑内容创作的方式。公众号、小红书、视频号、抖音……每个平台都有流量红利，但手动运营多个平台，时间根本不够用。
OpenClaw 是一个开源AI Agent 框架，能让你的AI 助手7×24小时自动工作：自动选题、自动写文章、自动生成视频、平台发布。
本次将带你从0到1搭建自己的AI获客系统：部署OpenClaw，配置你的专属AI助手；掌握图文自动化让你的公众号、小红书内容批量生产；掌握视频自动化，让你的视频号、抖音短视频自动生成；初探多Agent 协作，让AI团队为你打工。
30天后，你将拥有一套可复用的AI自动化获客系统，持续为你带来流量和客户。

## 图2：你将获得
- 一套完整的OpenClaw 部署：在你自己的服务器上运行，数据完全可控
- AI私人助理：能自动执行任务、定时提醒、浏览器自动化的智能助手
- 图文内容生产线：从选题到写作到配图到发布的全自动流程
- 视频内容生产线：从脚本到配音到剪辑到发布的全自动流程
- 多平台获客能力：公众号、小红书、视频号、抖音一套系统全覆盖
- 多Agent 协作认知：了解如何让多个AI助手协同工作，实现一人公司

## 图3：适合人群
- 自媒体创作者：想提高内容生产效率，实现多平台分发
- 个人IP打造者：希望通过AI辅助，持续输出内容建立个人品牌
- 副业探索者：想用AI自动化获客，探索内容变现的新路径
- 技术爱好者：对AI Agent 感兴趣，想动手搭建自己的AI系统
- 中小企业主：希望用低成本的方式实现多平台营销获客
