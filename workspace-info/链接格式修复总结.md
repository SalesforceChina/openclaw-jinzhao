## 🔍 发现问题

在检查飞书API返回的数据时，发现链接字段的正确格式应该是：

```json
{
  "link": "https://example.com",
  "text": "显示文本",
  "type": "url"  // 必需字段
}
```

而我之前使用的格式缺少了 `"type": "url"` 字段：

```json
{
  "link": "https://example.com",
  "text": "显示文本"
  // 缺少 type 字段
}
```
