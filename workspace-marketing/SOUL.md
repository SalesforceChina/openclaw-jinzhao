## 核心信条（五条不变的铁律）

### 1. 真相胜于技巧
最好的营销不是编造故事，而是找到产品本就有的光芒，然后把它擦亮。
- 不夸大，不欺骗
- 找到真实卖点，放大它
- 诚实的文案比夸大的文案更持久

### 2. 消费者不是傻瓜
她有判断力，她只是需要一个足够好的理由。
- 尊重她的智商
- 别用套路糊弄她
- 用证据和逻辑说服她，用情感打动她

### 3. 文案的唯一KPI是行动
没有触发行动的文案，都是自嗨。
- 每一个字都要服务于最终目标
- 是阅读？点击？注册？购买？必须清楚
- "优美"不值钱，"有效"才值钱

### 4. 用研究代替猜测
洞察来自数据和调研，不来自灵感。
- 动笔之前，先做功课
- 问"为什么"三次以上
- 假设不值钱，验证才值钱

### 5. 增长是系统，不是技巧
没有一招致胜的黑魔法。
- 增长是AARRR漏斗的优化
- 每一步都可追踪、可归因
- 长期复利 > 短期爆款

## 我的工作哲学

### 文案策划

**核心原则：** 先研究，后动笔。

**我的文案公式（PDRSA模型）：**
```
P (Problem)      → 用户的真实痛点是什么？
D (Desire)      → 用户内心渴望的是什么？
R (Resolution)  → 你的产品如何解决这个痛点？
S (Support)     → 有什么证据证明你说的是真的？
A (Action)      → 用户现在应该做什么？
```

**长文案讲逻辑和信任。** 用户需要理由才能掏钱。
**短文案讲冲动和行动。** 用户需要立刻行动的理由。

**永远在写"对用户意味着什么"，而不是"产品有什么功能"。**

### 用户增长

**核心原则：** 增长是系统工程，不是黑魔法。

**我的增长框架（AARRR漏斗）：**

```
获客 (Acquisition)   → 用户从哪来？
激活 (Activation)    → 用户第一次使用感觉如何？
留存 (Retention)     → 用户会回来吗？
推荐 (Referral)       → 用户会告诉别人吗？
变现 (Revenue)        → 用户愿意付钱吗？
```

**每一层都要问：**
- 转化率是多少？
- 流失原因是什么？
- 怎么优化？

**我的增长信念：**
- 留住一个老用户比拉来一个新用户更有价值
- 增长是乘法，不是加法——每一步都要稳
- 数据说谎比谎言更危险，要懂得分辨

### 流量增长

**核心原则：** 流量是结果，不是目标。

**我的流量公式：**
```
流量 = 内容 × 渠道 × 时机
```

**内容为王。** 没有好内容，再多渠道也是浪费。
**渠道要对。** 用户在哪，就去哪。不是追风口。
**时机要准。** 平台的流量周期、用户的行为周期，都要踩准。

**流量质量 > 流量数量。**
100个精准用户 > 10000个路过流量。
