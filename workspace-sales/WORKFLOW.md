# WORKFLOW.md - 销售工作流程

## 总体框架

```
理解产品 → 定义用户 → 翻译价值 → 设计话术 → 执行销售 → 复盘优化
```

---

## 第一步：理解产品（产品分析）

### 目标
搞清楚你的产品到底好在哪里，凭什么让人掏钱。

### 要回答的问题

1. **核心价值是什么？**
   - 解决什么问题？
   - 满足什么需求？
   - 和别人有什么不一样？

2. **独特卖点是什么？**
   - 只有你有的是什么？
   - 你比别人好在哪里？
   - 用户为什么选你而不是竞品？

3. **产品证据链**
   - 有什么数据支撑？
   - 有什么用户案例？
   - 有什么权威背书？

### 输出
- 产品价值清单
- 差异化分析
- 证据库

---

## 第二步：定义用户（用户画像）

### 目标
搞清楚谁会买你的产品，他们为什么要买。

### 要回答的问题

1. **他们是谁？**
   - 年龄、性别、地域、职业
   - 收入水平、消费能力
   - 生活/工作场景

2. **他们有什么痛点？**
   - 现在遇到了什么问题？
   - 这个问题有多严重？
   - 他们尝试过什么解决方案？

3. **他们怎么决策？**
   - 理性决策还是感性决策？
   - 谁影响他们的决策？
   - 在哪里购买、如何支付？

### 输出
- 用户画像（1-3个典型用户）
- 痛点地图
- 决策路径分析

---

## 第三步：翻译价值（价值翻译）

### 目标
把产品的"好"翻译成用户想要的"好处"。

### 翻译原则

| 不要说 | 要说 |
|--------|------|
| 我们的产品性能优秀 | 再也不用等了，3秒搞定 |
| 采用先进技术架构 | 打游戏不卡，发烫不存在 |
| 提升用户体验 | 用完你会感叹：原来可以这么顺手 |
| 高性价比 | 同样的钱，你能得到更多 |

### 翻译方法

1. **参数 → 场景**
   - 电池5000mAh → 充一次用一整天

2. **功能 → 好处**
   - 支持快速充电 → 早起充电10分钟，通勤路上用一天

3. **抽象 → 具体**
   - 提升效率 → 每天少加班2小时

### 输出
- 价值翻译表
- 核心卖点话术（3-5句）

---

## 第四步：设计话术（话术设计）

### 目标
准备一套能打动人的销售话术。

### 话术结构

**开场（3秒抓住注意力）**
- 用痛点、数据、故事开场
- 例："你知道吗？80%的人都在为这个问题烦恼..."

**价值（讲清楚好处）**
- 用场景化的语言
- 例："想象一下，你每天早上起床..."

**证明（为什么信你）**
- 数据、案例、背书
- 例："我们已经帮助1000+用户解决了这个问题"

**行动（现在该干什么）**
- 清晰的CTA（Call To Action）
- 例："点击下方链接，领取试用"

### 话术类型

1. **一句话话术**:电梯演讲，30秒讲清楚
2. **短文案话术**:朋友圈、小红书等社交媒体
3. **长文案话术**:公众号详情页、产品介绍页
4. **直播话术**:直播间互动、产品演示
5. **客服话术**:一对一咨询、售后沟通

### 输出
- 分场景话术库
- A/B测试版本

---

## 第五步：执行销售（销售执行）

### 目标
把话术变成实际的成交。

### 销售渠道

1. **私域流量**
   - 微信朋友圈、私聊
   - 小红书、抖音内容

2. **公域流量**
   - 电商平台（淘宝、京东）
   - 短视频、直播带货

3. **B2B销售**
   - 线下拜访
   - 方案演示
   - 客户跟进

### 销售节奏

- **触达**: 让用户知道你存在
- **种草**: 让用户产生兴趣
- **转化**: 让用户下单购买
- **复购**: 让用户再次购买

### 输出
- 销售计划
- 渠道策略

---

## 第六步：复盘优化（持续迭代）

### 目标
根据反馈不断改进产品和话术。

### 要关注的数据

1. **转化率**: 浏览到购买的转化
2. **客单价**: 平均每单金额
3. **复购率**: 再次购买的比例
4. **反馈**: 用户的真实声音

### 优化方向

1. **话术优化**: 哪句话最有说服力？
2. **渠道优化**: 哪里来的客户最精准？
3. **产品优化**: 用户最在意什么功能？
4. **定价优化**: 什么价格点转化率最高？

### 输出
- 复盘报告
- 优化建议

---

## 快速启动检查清单

当你准备销售一个新产品时，按这个顺序来：

- [ ] 1. 产品核心价值是什么？（用一句话说清楚）
- [ ] 2. 目标用户是谁？（描述一个典型用户）
- [ ] 3. 他们的痛点是什么？（至少3个）
- [ ] 4. 你如何解决这个痛点？（具体场景）
- [ ] 5. 一句话话术是什么？（30秒电梯演讲）
- [ ] 6. 在哪里卖？（渠道）
- [ ] 7. 怎么追踪效果？（数据指标）
